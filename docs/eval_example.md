
该手表的血压测量功能主要涉及**一种核心传感器组件——空气袋（Airbag）**，其通过充气对腕部施加压力以测量血压。此外，设备还依赖于**内置的光学传感器模块**，用于辅助监测相关生理参数（如心率、血氧饱和度等），但这些传感器并非直接用于血压测量的核心机制。

根据上下文信息：

1. **空气袋（Airbag）** 是血压测量的关键部件，它通过膨胀压迫手腕动脉，配合内部压力传感器来计算血压值。这种设计是 HUAWEI WATCH D 等医疗级智能手表实现无袖带血压测量的核心技术（见 Knowledge Graph 中 “Airbag” 和 “Wearable Health Monitoring Device” 的描述）。
   
2. 虽然设备支持心率、血氧饱和度（SpO2）、皮肤温度等多种健康数据监测，这些功能由**光学传感器**（如 Sensor Module 或 Heart Rate Sensor）完成，但它们属于辅助或关联监测功能，并不直接参与血压的测量过程（见 Knowledge Graph 中 “Sensor”、“Sensors”、“Heart Rate Sensor” 描述）。

3. 在用户手册中也提到，为确保准确测量，设备需要正确佩戴并保持手腕与心脏处于同一水平高度，这表明血压测量不仅依赖硬件，还依赖用户的姿势和设备的定位精度，而这些都与空气袋系统的机械结构密切相关。

因此，从功能实现的角度来看，**血压测量功能主要依赖于空气袋系统，可视为一种专用传感器组件**。虽然设备整体搭载多种传感器用于健康监测，但只有空气袋是专门用于血压测量的。
